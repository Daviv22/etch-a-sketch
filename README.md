# Etch-a-Sketch

Um Etch-a-Sketch interativo no navegador, inspirado no brinquedo clássico. Desenhe livremente em uma grade ajustável usando o mouse ou toque na tela.

🎨 **[Acesse o projeto ao vivo](https://daviv22.github.io/etch-a-sketch/)**

## Funcionalidades

- **Desenho livre** — clique e arraste para pintar as células da grade
- **Seletor de cor** — escolha qualquer cor para o seu pincel
- **Borracha** — alterne para o modo borracha e apague partes do desenho
- **Limpar** — reinicia a grade com um clique
- **Tamanho ajustável** — slider para redimensionar a grade de 1×1 até 64×64

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Como executar localmente

Não há dependências ou build necessário. Basta clonar o repositório e abrir o `index.html` no navegador:

```bash
git clone https://github.com/seu-usuario/etch-a-sketch.git
cd etch-a-sketch
```

Abra o arquivo `index.html` diretamente no navegador, ou use uma extensão como o **Live Server** no VS Code.
